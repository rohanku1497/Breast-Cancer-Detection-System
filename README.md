# Breast-Cancer-Detection-System
Machine learning project that classifies breast tumors as benign or malignant using Logistic Regression. Includes data preprocessing, EDA, feature scaling, model evaluation, and performance visualization on the Wisconsin Breast Cancer dataset.

## Dataset Information

This project uses the Breast Cancer Wisconsin (Diagnostic) Dataset from the UCI Machine Learning Repository.

### Dataset Overview
- Instances: 569
- Features: 30 numerical features
- Task: Binary Classification
- Target Variable:
  - Malignant (M) – Cancerous tumor
  - Benign (B) – Non-cancerous tumor
- Missing Values: None

### Feature Description
The features are computed from digitized images of fine needle aspirate (FNA) samples of breast masses and describe characteristics of cell nuclei, including:

- Radius
- Texture
- Perimeter
- Area
- Smoothness
- Compactness
- Concavity
- Concave Points
- Symmetry
- Fractal Dimension

For each characteristic, the dataset contains measurements such as mean, standard error, and worst (largest) value, resulting in 30 predictive features.
